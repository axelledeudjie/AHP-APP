Step 1 : Define Criteria

Add at least 3 criteria to evaluate your alternatives. Numerical criteria use raw scores (higher = better). Categorical criteria use an ordered preference scale you define (worst → best).

Step 2 : Define Alternatives

Add at least 2 alternatives you want to choose between.

Step 3 : Score Alternatives

Rate each alternative per criterion. For numerical criteria: enter a positive number (higher = better). For categorical criteria: select a category.

Step 4 : Pairwise Comparison of Criteria

For each pair, select how important the left criterion is relative to the right criterion. Values > 1 favour the left; values < 1 favour the right.
